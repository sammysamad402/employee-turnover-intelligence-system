
# Employee Turnover Intelligence System

A portfolio-grade HR Analytics and Employee Attrition Prediction project.

## Highlights
- End-to-End ML Pipeline
- SMOTE Balancing
- KMeans Employee Segmentation
- Hyperparameter Optimization
- ROC-AUC Benchmarking
- Risk Scoring Engine
- Business Recommendations

## Tech Stack
Python, Pandas, Scikit-Learn, Seaborn, Matplotlib, Imbalanced-Learn

## Business Impact
Predict employee turnover and identify intervention opportunities before resignation occurs.
